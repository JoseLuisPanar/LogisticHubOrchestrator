# LogisticsHubOrchestrator

Plataforma de gestión logística.

## Stack tecnológico
- **Backend**: C# / .NET 10 / Clean Architecture / ASP.NET Core REST API
- **Frontend**: React 18 + Vite 8
- **Base de datos**: PostgreSQL 17/18
- **ORM**: Entity Framework Core 10
- **Autenticación**: JWT + ASP.NET Core Identity

## Estructura del repositorio
```
src/
  backend/   → Solución .NET (Domain, Application, Infrastructure, Api)
  frontend/  → SPA React + Vite
docs/
  SETUP.md            → Arranque en local
  ARCHITECTURE.md      → Convenciones de arquitectura y código
  COWORK_RULES.md       → Reglas y límites de trabajo para Cowork
```
